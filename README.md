# SLNS Website

A professional Express-based website for SLNS Document Writers.

## Features
- Responsive one-page layout
- Contact form endpoint
- Security middleware with Helmet and rate limiting
- Static asset optimization
- Local development server

## Run locally
```bash
npm install
npm start
```

Open http://localhost:3000

## Deploy
This project is ready to be deployed to Render, Railway, or any Node.js hosting provider.
